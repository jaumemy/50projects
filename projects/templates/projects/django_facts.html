Django Facts

What is Django

Django is a Python-based free and open-source web framework that follows the model-template-views.
Django's primary goal is to ease the creation of complex, database-driven websites with as little code as posible.
As they say: "Django,the web framework for perfectionists with deadlines".

Who created it and when

First release was in 21 July 2005, by Adrian Holovaty and Simon Wilson, both journalists with web programming skills working at the Lawrence Journal-World, in Kansas, United States.

Who uses Django

Some sites that use Django are The Washington Post, Instagram, Spotify, Youtube, Dropbox, Prezi or the NASA official website.

Community

There is a semiannual conference for Django developers and users, named "DjangoCon", that has been held since September 2008, usually in Europe, but also in the United States.
Django has spawned user groups and meetups around the world, the most notable group is the Django Girls organization, which began in Poland but now has had events in 91 countries.

Tell me more

Django is named after the jazz guitarist and composer Django Reindhart.
You can find more info in the Official Django Website or the Wikipedia Page.
